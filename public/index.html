Bem-vindo a SisCaju!

Os “endpoints das APIs” já implementadas permitem realizar as operações CRUD (Create, Read, Update e Delete) na base de dados. Esta aplicação (backend) está acessível a partir do seguinte domínio:
https://pwa-siscaju.herokuapp.com/

Para realizar as operações ‘Create’ e ‘Update’, é ainda recomendado utilizar uma ferramenta para testar as APIs, como o Postman ou outro similar.

1. CRUD para utilizadores do SisCaju:

Operation name Method Endpoint Query values Description
login() POST /login N/A
getUsers() GET /users N/A
getUsers() GET /users role | province | district | admin-post
addUser() POST /users N/A
getUserById() GET /users/:userId N/A
updateUser() PATCH /users/:userId N/A
deleteUser() DELETE /users/:userId N/A
