Bem-vindo a SisCaju!

Os “endpoints das APIs” já implementadas permitem realizar as operações CRUD (Create, Read, Update e Delete) na base de dados. Esta aplicação (backend) está acessível a partir do seguinte domínio:
https://pwa-siscaju.herokuapp.com/

Para realizar as operações ‘Create’ e ‘Update’, é ainda recomendado utilizar uma ferramenta para testar as APIs, como o Postman ou outro similar.

1. CRUD para utilizadores do SisCaju:

Método Endpoint Descrição
GET /users Leitura de todos os utilizadores já registados
POST /users Registo de um novo utilizador
UPDATE /users/:userId Atualização de um utilizador já registado
DELETE /users/:userId Eliminação de um utilizador já registado

2. CRUD para produtores do caju:

Método Endpoint Descrição
GET /farmers Leitura de todos os produtores já registados
POST /farmers Registo de um novo produtor
GET /farmers/:farmerId Leitura de um produtor, usando o seu ID como parâmetro
UPDATE /farmers/:farmerId Atualização de dados de um produtor, usando o seu ID como parâmetro
DELETE /farmers/:farmerId Eliminação de um produtor já registado

3. CRUD para pomares do caju e suas respetivas subdivisões:

Método Endpoint Descrição
GET /farmlands Leitura de todos os pomares já registados
GET /farmlands/:farmlandId Leitura de um pomar, usando o ID do pomar como parâmetro.
GET /farmlands/:farmerId Leitura dos pomares pertencendo ao produtor, usando o ID do produtor como parâmetro
POST /farmlands/:farmerId Registo do pomar pertencendo ao produtor, usando o ID do produtor como parâmetro
DELETE /farmers/:farmerId Eliminação de um produtor já registado
